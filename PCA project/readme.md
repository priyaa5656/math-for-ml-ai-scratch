# 🧠 PCA Project — Dimensionality Reduction
## 📌 What is this project?
This project uses **PCA (Principal Component Analysis)** to reduce high-dimensional data into lower dimensions and visualize it.

---

## 🎯 Goal
- Convert 4D data → 2D  
- Preserve important information  
- Visualize patterns using graph  
---

## ⚙️ How it works

1. Load dataset (Iris 🌸)  
2. Apply PCA  
3. Reduce dimensions  
4. Plot graph  
---

## 📊 Output

- 3 clusters:
  - Setosa  
  - Versicolor  
  - Virginica  
👉 Data becomes easy to understand visually  
---

## 🚀 How to Run

```bash
pip install matplotlib scikit-learn
python code.py
```
